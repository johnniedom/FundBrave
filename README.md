### Prerequisites
- Node.js 18+
- Python 3.9+
- PostgreSQL 14+
- Redis 7+

### Installation

```bash
# Clone the repository
git clone https://github.com/FundBrave/FundBrave.git
cd fundbrave

# Run the setup script
chmod +x scripts/setup.sh

# Install dependencies for all packages
```
