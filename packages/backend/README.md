# FundBrave Backend

Comprehensive NestJS GraphQL API for the FundBrave decentralized fundraising platform.

## Architecture Overview

The backend provides:
- **GraphQL API** (Apollo Server) for all platform operations
- **Real-time WebSocket Events** (Socket.IO) for live updates
- **Blockchain Event Indexing** for syncing on-chain data
- **PostgreSQL Database** with Prisma ORM
- **JWT + SIWE Authentication** for Web3 users
